Business Context

a) Only the Information Supervisor can access the Information pool;
b) The Information Requestor should be able to query the status of his request at any time, and the system should be able to indicate the process the request is in at that particular time;
c) The system should not indicate the user the task is with but should only report on the process the request is in;
d) The database is updated all the time when a change happens;
e) Audit trails are kept at all times i.e. (on Create, Update and Delete);
f) The tracking number format for Information Provision is in the following format TRC_’UserID’_’CCYYMMDD’ e.g. ‘TRC_1234_20151020’
